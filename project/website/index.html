About "The Voxel Reactor"

This website explores my creative coding journey through the use of the P5.js Library to create Art, Visuals and much more. Here, you can find my visual style which integrates colours, audio-reactivity & self-generativity.

My journey has culminated into the realization of "The Voxel Reactor"; An Audio-Reactive Visualizer that features a Voxelized Polygon which grows with the Audio.

This is accompanied by the various Audio-Reactive Landscapes which can be called using your keyboard number keys 1-9.

Have a look at my process log, my p5js sketches, and github repo.
